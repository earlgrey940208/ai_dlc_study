# User Personas

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| **이름** | 김손님 (대표 고객) |
| **역할** | 매장 방문 고객 |
| **목표** | 빠르고 편리하게 음식 주문 |
| **사용 환경** | 테이블 태블릿 (브라우저) |
| **특징** | 별도 로그인 없이 즉시 사용, 터치 기반 조작 |
| **Pain Point** | 직원 호출 대기 시간, 복잡한 주문 과정 |

### 주요 행동
- 메뉴를 카테고리별로 탐색
- 장바구니에 담고 수량 조절
- 주문 후 상태 확인

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| **이름** | 이주인 (매장 사장님) |
| **역할** | 매장 운영 전반 관리 (단일 관리자) |
| **목표** | 실시간 주문 파악, 효율적 매장 운영 |
| **사용 환경** | PC 또는 태블릿 브라우저 |
| **특징** | 모든 관리 기능 수행 (주문 모니터링, 테이블 관리) |
| **Pain Point** | 주문 누락, 테이블 상태 파악 어려움 |

### 주요 행동
- 실시간 주문 대시보드 모니터링
- 주문 상태 변경 (대기중 → 준비중 → 완료)
- 테이블 세션 관리 (이용 완료 처리)
