# 테이블오더 서비스 - 요구사항 정의서

## Intent Analysis

- **User Request**: 테이블오더 서비스 MVP 개발
- **Request Type**: New Project (Greenfield)
- **Scope Estimate**: Multiple Components (Frontend + Backend + DB)
- **Complexity Estimate**: Moderate

---

## 기술 스택 결정사항

| 영역 | 기술 | 비고 |
|------|------|------|
| Frontend | React | SPA 방식 |
| Backend | Spring Boot (Java) | REST API |
| Database | PostgreSQL | 관계형 DB |
| 실시간 통신 | SSE (Server-Sent Events) | 단방향 실시간 |
| 배포 환경 | Docker Compose | 로컬 개발 환경 |
| 이미지 관리 | 외부 URL 입력 방식 | 업로드 없음 |

## Functional Requirements

### FR-1: 고객용 기능

#### FR-1.1: 테이블 태블릿 자동 로그인 및 세션 관리
- 관리자가 1회 초기 설정 (매장 식별자, 테이블 번호, 비밀번호)
- 로그인 정보 로컬 저장 후 자동 로그인

#### FR-1.2: 메뉴 조회 및 탐색
- 카테고리별 메뉴 분류 및 표시
- 메뉴 상세 정보 (메뉴명, 가격, 설명, 이미지)
- 카드 형태 레이아웃, 터치 친화적 UI (최소 44x44px)

#### FR-1.3: 장바구니 관리
- 메뉴 추가/삭제, 수량 조절
- 총 금액 실시간 계산
- 클라이언트 측 로컬 저장 (새로고침 시 유지)
- 서버 전송은 주문 확정 시에만

#### FR-1.4: 주문 생성
- 주문 내역 최종 확인 후 확정
- 성공 시: 주문 번호 표시 → 장바구니 비우기 → 메뉴 화면 자동 리다이렉트 (5초)
- 실패 시: 에러 메시지 표시, 장바구니 유지

#### FR-1.5: 주문 내역 조회
- 현재 테이블 세션 주문만 표시
- 주문 시간 순 정렬, 상태 표시 (대기중/준비중/완료)

### FR-2: 관리자용 기능

#### FR-2.1: 매장 인증
- 매장 식별자 + 사용자명 + 비밀번호 입력
- JWT 토큰 기반, 16시간 세션 유지

#### FR-2.2: 실시간 주문 모니터링
- SSE 기반 실시간 주문 업데이트 (2초 이내)
- 테이블별 그리드/카드 레이아웃
- 주문 상태 변경 (대기중 → 준비중 → 완료)
- 신규 주문 시각적 강조

#### FR-2.3: 테이블 관리
- 테이블 초기 설정 (번호, 비밀번호, 16시간 세션)
- 주문 삭제 (확인 팝업, 총 주문액 재계산)
- 테이블 이용 완료 (세션 종료, 주문 내역 이력 이동, 리셋)
- 과거 주문 내역 조회 (날짜 필터링)

#### FR-2.4: 메뉴 관리
- **MVP 범위 외**: Seed data로 초기 데이터 처리
- 메뉴 CRUD는 MVP에 포함하지 않음

## Non-Functional Requirements

### NFR-1: 성능
- SSE 주문 알림 2초 이내 전달
- 테이블 규모: 소규모 (1~10개)

### NFR-2: 보안
- JWT 토큰 기반 인증
- 비밀번호 평문 저장 금지 (기본 해싱 적용)
- 고급 보안 기능은 향후 브라운필드에서 강화 예정

### NFR-3: 배포
- Docker Compose로 로컬 환경 구성 (필수 - 로컬 실행 보장)
- Rancher를 통한 배포 시도 (선택 - 실패 시 로컬 실행으로 대체)
- Frontend, Backend, PostgreSQL 컨테이너 구성

### NFR-4: UI/UX
- 터치 친화적 인터페이스 (최소 44x44px)
- 카드 형태 레이아웃
- 명확한 시각적 계층 구조

---

## 제외 사항 (Out of Scope)

- 결제 처리 (PG사 연동, 영수증 등)
- 복잡한 인증 (OAuth, SNS, 2FA)
- 이미지 업로드/리사이징
- 알림 시스템 (푸시, SMS, 이메일)
- 주방 기능 (주방 전달, 재고 관리)
- 고급 기능 (매출 리포트, 직원 관리, 예약, 리뷰, 다국어)
- 외부 연동 (배달 플랫폼, POS, 소셜 미디어)

## MVP 범위 요약

### 고객용
- 테이블 자동 로그인 및 세션 관리
- 메뉴 조회
- 장바구니 관리
- 주문 생성
- 주문 내역 조회 (현재 세션)

### 관리자용
- 매장 인증 (16시간 세션)
- 실시간 주문 모니터링 (SSE, 그리드 레이아웃)
- 테이블 관리 (초기 설정, 주문 삭제, 이용 완료, 과거 내역)
- 메뉴 관리: Seed data만 (CRUD 미포함)
